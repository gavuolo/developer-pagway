# developer-pagway

---
<h2>Instalação</h2>

1. Clonar o Projeto
```bash
git clone git@github.com:gavuolo/projeto22-autoral-back-end.git
```

2. Intalar dependências
```bash
npm install
```

3. Criar DB dev
```bash
npm run dev:migrate
```

4. Criar DB test
```bash
npm run test:migrate
```

3. Rodar o projeto
```bash
npm run dev
```
